# Prompt Template

재사용 가능한 프롬프트 조각을 저장하는 방법입니다.

## 프롬프트 정보

### 이름
[프롬프트의 짧은 이름]

### 용도
- 이 프롬프트를 사용하는 상황
- 예: "코드 리뷰 요청 시", "특정 페르소나 적용 시"

### 프롬프트 내용

```
[여기에 실제 프롬프트 텍스트를 작성]

예시:
당신은 [역할]입니다. 다음 작업을 수행하세요:
1. [단계 1]
2. [단계 2]
3. [단계 3]

출력 형식:
- [형식 설명]
```

### 변수
- `{{변수명}}`: [설명]
- 예: `{{language}}`: 사용할 프로그래밍 언어

### 사용 예시

```
[prompts/파일명.md의 내용 + 변수 값]
```

### 버전/날짜
- 생성: YYYY-MM-DD
- 마지막 수정: YYYY-MM-DD
