## Python
- Follow **PEP8** style guide (snake_case)

## GDScript (Godot)
- **변수/함수**: snake_case
- **클래스/파일**: PascalCase
- **상수**: UPPER_SNAKE_CASE
- Godot 공식 스타일 가이드 준수

## JavaScript
- Use **camelCase** for naming conventions

**Why:** User prefers Godot/GDScript standard naming conventions.

**How to apply:** When writing GDScript code, use snake_case for variables/functions, PascalCase for classes and file names.
