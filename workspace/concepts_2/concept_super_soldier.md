# 컨셉: 슈퍼솔져

## 핵심 컨셉
바이올렛 에버가든에서 나오는 주인공이나 건슬링어 걸즈의 소녀들을 모티프로 삼아, 판타지 셋팅의 전장에서 얻은, 감정이 옅은 슈퍼솔져 소녀를 어떻게 다룰 것인지 결정하는 게임. Papers Please 느낌의, 정해진 스토리 내에서 플레이어가 반복적인 행동을 하며 점점 더 어려운 / 흥미로워지는 결정을 내리며 얻기 쉽지 않은 엔딩들에 도달하게 하는 게임.

## 게임플레이
- **전투**: 선택지로 방침을 결정하고, 라이브 틱커의 형태로 메시지 로그가 업데이트되는 식.
- **스케줄 관리**: 스케줄 형식으로 식사를 주거나, 소녀를 훈련시킬 수 있음.
- **이벤트**: 점령지 마을에서 맛있는 걸 사 줄 수도 있고, 다양한 선택지를 통해 소녀와의 관계 및 전투 결과가 변화.
- 플레이어가 스토리에 나오는 여러 선택지들을 잘 진행하면 소녀를 전장에 많이보내지 않고도 클리어할 수 있음.
- 스토리 게임으로 제작.

## 캐릭터
- **슈퍼솔져 소녀**: 감정이 옅은 슈퍼솔져. 전장에서 얻은 존재. 플레이어의 선택에 따라 서서히 변화하거나, 혹은 그대로 머물 수 있음.

## 판매 전략
최대 10달러에 책정하여 빠르게 완성 후 판매. 부족하더라도 완성하여 판매하고, 개선하여 후속작을 내는 방식.

---

## 평가

### 1. 2개월 혼자 완성 가능성: A
- 단일 메인 캐릭터(슈퍼솔져 소녀)에 집중하므로 아트 리소스가 극도로 절약됨.
- 전투는 "선택지 → 메시지 로그 업데이트"라 단순한 텍스트 시뮬레이션. 실시간 전투 시뮬레이션이 아님.
- 스케줄 관리(식사, 훈련, 선물)는 Papers Please의 일상 루틴과 유사하게 단순한 선택지로 구현 가능.
- 스토리 분기는 있으나, 단일 캐릭터와의 관계 중심이므로 컨셉 1(벙커)보다 분기 수가 적음.

### 2. 참신함 / 인터랙티브함: A
- "감정이 옅은 슈퍼솔져 소녀를 다룬다"는 설정은 매우 흥미롭고 참신함.
- Papers Please의 "단순한 작업이 도덕적/감정적 딜레마가 됨"과 유사한 인터랙티브 서사.
- 바이올렛 에버가든의 "감정을 배워가는 존재" + 건슬링어 걸즈의 "소녀+전쟁"이라는 모티프를 게임으로 재현하는 것은 차별화됨.
- 플레이어의 선택이 소녀의 감정/운명에 미치는 영향을 직접적으로 보여주는 구조.

### 3. 귀여운 여자애들 / 재미있는 대사 적합성: A
- 슈퍼솔져 소녀가 메인 캐릭터이므로 사용자의 아트 강점을 100% 발휘 가능.
- "감정이 옅다"는 설정은 오히려 대사의 변화(초기의 무감정 → 점진적인 변화)를戏적으로 보여줄 수 있음.
- 식사, 훈련, 선물 등의 일상적 선택에서 나오는 소소한 대사가 캐릭터의 성장을 느끼게 함.

### 4. 비주얼 노벨 엔진 연계성: A
- 전투가 "선택지 + 메시지 로그"라면 VN 엔진의 기본 기능(선택지, 텍스트 출력)으로 충분히 구현 가능.
- 스케줄 관리도 VN의 선택지/분기 구조와 동일.
- 추가 UI가 거의 필요 없음. Papers Please의 문서 검사 UI보다 훨씬 단순함.

### 5. 최소 프로그래밍 필요도: 낮음~중간
- 메시지 로그 업데이트, 스케줄 선택, 엔딩 분기 조건 등은 VN 엔진의 기본 기능으로 커버 가능.
- 단순한 수치 계산(전투 결과 = 선택지 + 소녀 능력치 + 상태) 정도만 추가.

### 6. $10 판매 적합성: A
- 단편 스토리 게임으로서 $5~$10 가격대에 매우 적합함.
- "감정이 옅은 슈퍼솔져 소녀"라는 설정 자체가 흥미를 끌어 구매 전환에 효과적임.
- 엔딩 분기(소녀를 많이보내지 않고 클리어 vs. 다른 엔딩)가 반복 구매/플레이 유도.

### 7. 후속작/시리즈 전개 가능성: A
- 한 명의 슈퍼솔져 소녀를 다룬 후, "다른 전장에서 다른 소녀를 만난다"는 식으로 후속작을 내기 쉬움.
- 세계관을 확장하거나, 같은 소녀의 이후 이야기를 다룰 수 있음.

### 종합
**5개 컨셉(원본) 및 4개 신규 컨셉을 통틀어 가장 균형 잡힌 선택.** Papers Please급 참신함을 확보하면서도, 프로그래밍 부담은 최소화. 단일 캐릭터에 집중하므로 아트/시나리오 리소스를 극도로 절약할 수 있음. 비주얼 노벨 엔진과 완벽하게 호환되며, 2개월 내 혼자 완성이 실제로 가능함. $10 가격대에 완벽하게 맞고, 후속작 전개도 용이함. **최우선 추천**.
