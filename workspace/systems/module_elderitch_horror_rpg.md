# 모듈: Eldritch Horror 기반 RPG 시스템

> **독립 모듈 문서**. 어떤 게임에도 삽입 가능한 턴제 RPG 메카닉.
> 
> 원작: 보드게임 『Eldritch Horror』
> 핵심: 주사위 풀 판정, 이중 전투, 뒷면 카드 상태이상, 징조 시스템

---

## 1. 개요

### 도입 목적
- 의미 없는 숫자 올리기(레벨업) 제거
- 기계적 MP 소모 스킬 제거  
- 단순 수치 뻥튀기 장비 제거
- 모든 선택/장비/스킬이 **서사와 생존에 직결**되는 시스템 구축

### 핵심 철학
**"하이 리스크 하이 리턴"**
- 강한 적을 상대하려면 피를 흘리거나 정신을 잃을 각오 필요
- 방어력이 데미지를 0으로 만드는 뻔한 전개 불가능
- 플레이어의 모든 판정은 생존과 직결

## 2. 능력치 시스템 (Attributes)

### 5대 핵심 능력치

| 능력치 | 영문명 | 주요 용도 | 전투 용도 |
|--------|--------|----------|----------|
| **지식** | Lore | 고대 유물, 마법, 비밀 문서 | 마법 사용, 약점 파악 |
| **영향력** | Influence | NPC 대화, 협상, 설득 | 적 협상, 구명 타협 |
| **관찰력** | Observation | 함정 발견, 숨은 문, 추적 | 선제공격, 도주, 정보 |
| **근력** | Strength | 물리 장애물, 도구 | 물리 공격, 방어 |
| **의지력** | Will | 공포 저항, 유혹 거부 | 공포 판정(필수) |

### 능력치 설계 가이드

```yaml
초기치 분배:
  총합: 12~15점 (게임 밸런스에 따라 조정)
  
  예시 캐릭터 A (전사형):
    근력: 4
    의지: 3
    관찰: 2
    지식: 1
    영향력: 2
    
  예시 캐릭터 B (마법사형):
    지식: 4
    의지: 3
    영향력: 2
    관찰: 2
    근력: 1

성장:
  전통적 레벨업 없음
  대신: 장비/아이템으로 주사위 추가
  또는 특정 이벤트로 능력치 1 영구 증가 (극히 드묾)
```

## 3. 주사위 풀 판정 시스템 (Dice Pool)

### 기본 판정

```
판정 절차:
1. 행동에 필요한 능력치 확인
2. 해당 능력치 수치만큼 6면체(D6) 굴림
3. 결과 5 또는 6이면 "성공(Success)"
4. 성공 개수에 따라 결과 결정

성공률 참고:
  1주사위: 33% (5-6)
  2주사위: 56% (1개 이상 성공)
  3주사위: 70%
  4주사위: 80%
  5주사위: 87%
```

### 결과 해석

| 성공 수 | 결과 | 설명 |
|--------|------|------|
| 0 | **실패** | 목표 실패 + 대가 발생 |
| 1 | **부분 성공** | 목표 달성 + 대가 동반 |
| 2 | **성공** | 목표 완전 달성 |
| 3 | **완벽한 성공** | 목표 달성 + 소소한 보너스 |
| 4+ | **대성공** | 목표 달성 + 의외의 이득 |

### 난이도 목표치 (Difficulty)

```yaml
대체 판정 방식 (성공 수 vs 목표치):
  쉬움(Easy): 1 성공 필요
  보통(Normal): 2 성공 필요  
  어려움(Hard): 3 성공 필요
  극악(Extreme): 4 성공 필요

사용 예시:
  "고대의 문을 연다"
  → 지식 판정 (3 성공 필요)
  → 지식 4점 → D6×4 굴림
  → [2, 4, 5, 6] → 2성공
  → 부분 성공: "문은 열렸지만 경고음이 울렸다"
```

### 보정 요소

```yaml
장비 효과 (모두 "+주사위" 형태):
  "예리한 단검": 전투 근력 판정 +1 주사위
  "방패": 피해 감소 1 (고정값, 주사위 아님)
  "발광석": 어두운 곳 관찰력 +2 주사위
  "고대의 반지": 지식 판정 +1 주사위

상태이상 효과:
  "부상": 근력 판정 -1 주사위
  "공포": 의지력 판정 -1 주사위
  "집중": 다음 판정 +2 주사위 (1회용)
  
주의: 최소 1주사위 보장 (음수 방지)

환경 효과:
  "어두움": 관찰력 -1, 근력 -1
  "좁은 공간": 근력 -1, 회피 불가
  "고대의 힘": 지식 +1, 의지력 -1
```

## 4. 이중 전투 시스템 (Dual-Phase Combat)

> **모든 전투는 반드시 정신력 판정 → 체력 판정 순서로 진행**

### 전투 흐름

```yaml
적 조우:
  1단계: 공포 판정 (Horror Check) - 필수
    능력치: 의지력 (Will)
    계산: 적 "공포값" - 플레이어 성공 수 = 정신력 피해
    
    예시:
      의지력 2 → D6×2 = [3, 6] → 성공 1개
      적 공포값 3
      정신력 피해: 3 - 1 = 2
      → "당신의 이성이 그 존재에 노출되었다"

  2단계: 전투 판정 (Combat Check)
    능력치: 근력 (Strength)
    계산:
      플레이어 성공 수: S
      적 전투값: C
      
      플레이어 피해: max(0, C - S)
      적 피해: S
      
    예시:
      근력 3 + 장비 +1 → D6×4
      [1, 3, 5, 6] → 성공 2개
      적 전투값 4
      
      플레이어 피해: 4 - 2 = 2
      적 피해: 2

  3단계: 전투 지속/종료
    - 적 체력 0 → 전투 종료, 승리
    - 적 생존 → 다음 턴 (플레이어 선택 → 적 반응)
    - 플레이어 체력/정신력 0 → 전투 불능
```

### 적 데이터 템플릿

```yaml
적_템플릿:
  ID: "고유식별자"
  이름: "표시명"
  체력: 숫자
  공포값: 숫자 (1~5, 높을수록 정신적 위협 큼)
  전투값: 숫자 (1~6, 높을수록 물리적 위협 큼)
  특성: [특수 능력 목록]

예시 적:
  고블린:
    체력: 3
    공포값: 1
    전투값: 2
    특성: ["겁쟁이"]
    
  그림자_괴물:
    체력: 8
    공포값: 4  # 강한 정신적 위협
    전투값: 3
    특성: ["공포유발", "회피"]
    
  늑대:
    체력: 5
    공포값: 2
    전투값: 3
    특성: ["무리", "추적"]
```

### 보스 전투 (다단계)

```yaml
보스_템플릿:
  페이즈: N
  
  페이즈별_스펙:
    페이즈1:
      체력: X
      공포값: Y
      전투값: Z
      특성: [특성목록]
      전환조건: 체력 N% 이하 또는 특정 행동
      
    페이즈2:
      체력: X'
      공포값: Y'
      전투값: Z'
      특성: [강화된 특성]

예시: "랫맨 군주"
  페이즈1: "경계"
    체력: 10, 공포값 3, 전투값 4
    특성: "첫 전투판정 플레이어 -1주사위"
    전환: 체력 7 이하
    
  페이즈2: "격분"  
    체력: 7, 공포값 4, 전투값 5
    특성: "매 턴 플레이어 피해 +1"
    전환: 체력 3 이하
    
  페이즈3: "최후의 발악"
    체력: 3, 공포값 5, 전투값 6
    특성: "1HP시 즉사급 공격 1회"
```

## 5. 뒷면 카드 상태이상 시스템 (Condition Cards)

> **모든 상태이상은 뒷면이 가려진 카드 형태**
> **징조(Reckoning) 발생시 뒷면 확인**

### 상태이상 카드 구조

```yaml
상태이상_카드_템플릿:
  ID: "유니크ID"
  이름: "표시명"
  유형: 물리적 | 정신적 | 특수
  
  앞면 (Front):
    효과: [지속 효과 목록]
    설명: "플레이어에게 보이는 설명"
    
  뒷면 (Back):
    징조조건: "발생 조건 설명"
    판정:  # 징조 발생시 수행할 판정
      능력치: 능력치명
      목표: 필요 성공 수
    결과:
      성공N+:
        효과: "회복 또는 경감"
      성공N-1:
        효과: "유지"
      실패(0):
        효과: "악화 또는 추가 피해"
    설명: "뒷면 텍스트 (긴장감 있는 묘사)"
```

### 징조(Reckoning) 시스템

```yaml
징조_발생_타이밍:
  - 전투 종료 후
  - 던전 귀환시
  - 특정 이벤트 후 (마법 사용, 저주 아이템 등)
  - 보스 전투 시작시
  - 게임 내 "징조 페이즈" (턴 기반)

징조_진행:
  1. 플레이어의 모든 상태이상 카드 뒷면 확인
  2. 각 카드별로 판정 수행
  3. 결과 적용 (회복/유지/악화)
  
징조_UI_예시:
  [징조 단계]
  당신의 상처들이 시험받습니다...
  
  [상처] 근력 판정 중... [성공 2개] → 회복!
  [공포] 의지력 판정 중... [성공 0개] → "공황"으로 악화
```

### 물리적 상태이상

```yaml
상처 (Wound):
  앞면:
    - 근력 판정 -1 주사위
    - 전투시 공포값 +1 (고통으로 인한 공포)
  뒷면:
    판정: 근력, 목표 2성공
    성공2+: 회복
    성공1: 유지
    실패: 체력 -3, "맹독" 추가

출혈 (Bleeding):
  앞면:
    - 매 턴 시작 체력 -1
    - 근력 -1 주사위
  뒷면:
    판정: 의지력, 목표 1성공
    성공: 출혈 정지
    실패: 체력 -2, "쇼크" 추가

골절 (Fracture):
  앞면:
    - 근력 -2 주사위
    - 이동시 추가 턴 소모
  뒷면:
    판정: 근력, 목표 3성공
    성공: "상처"로 경감
    실패: "불구"로 악화

맹독 (Severe Poison):
  앞면:
    - 매 턴 체력 -2
    - 근력 -1 주사위
  뒷면:
    판정: 지식, 목표 2성공
    성공: 해독
    실패: 체력 -5, "마비" 추가
```

### 정신적 상태이상

```yaml
공포 (Terror):
  앞면:
    - 의지력 -1 주사위
    - 공포 판정시 공포값 +1
  뒷면:
    판정: 의지력, 목표 2성공
    성공: 회복
    실패: "공황"으로 악화

공황 (Panic):
  앞면:
    - 전투 선택지 1개 무작위 제거
    - 의지력 -2 주사위
  뒷면:
    판정: 의지력, 목표 3성공
    성공: "공포"로 경감
    실패: 강제 도주, 아이템 분실

집착 (Obsession):
  앞면:
    - 특정 행동 강제
    - 영향력 -1 주사위
  뒷면:
    판정: 의지력, 목표 2성공
    성공: 회복
    실패: 정신력 -3, 강제 행동
```

### 특수 상태이상

```yaml
저주 (Curse):
  앞면:
    - 매 탐사 시작 1D6=1시 악 이벤트
    - 모든 판정 -1 주사위
  뒷면:
    판정: 지식, 목표 3성공
    성공: 해제
    실패: 최대 정신력 -1 (영구)

계약 (Pact):
  앞면:
    - 특정 혜택 (능력치 +1, 무료 아이템 등)
  뒷면:
    판정: 의지력, 목표 4성공 (극악)
    성공: 파기, 혜택 회수
    실패: 체력/정신력 절반 손실 또는 사망

표식 (Mark):
  앞면:
    - 특정 적 우선 타겟
    - 공포값 +1
  뒷면:
    특수 이벤트 발생 (추격, 암살)
    또는 "저주"로 악화
```

## 6. 장비 및 마법 시스템

### 장비 원칙

```yaml
장비는 영구적 스탯 상승이 아닌 "+주사위" 형태:
  "강철 검": 전투 근력 +1 주사위
  "은빛 단검": 유령과 전투시 근력 +2 주사위
  "방패": 피해 감소 1 (고정값)
  "발광석": 어두운 곳 모든 판정 +1 주사위

장비 파손:
  - 전투 실패시 1D6=1이면 장비 파손
  - 파손시 효과 절반 또는 사용 불가
```

### 마법 시스템

```yaml
마법_사용_절차:
  1. 지식 판정 (발동)
  2. 성공시 효과 발생
  3. 마법 카드 뒷면 확인 (징조 효과)

마법_템플릿:
  이름: "화염의 손길"
  발동: 지식 판정 2성공 필요
  효과: 적 체력 -4 즉시
  
  뒷면:
    판정: 지식, 목표 2성공
    성공2+: 아무 일 없음
    성공1: 정신력 -1 (대가)
    실패: "과열" - 체력 -2, 아이템 파손
```

## 7. 통합 예시: 전투 세션

```
=== 그림자 괴물과의 전투 ===

[플레이어 상태]
체력: 8/10 | 정신력: 6/8
능력치: 근력 3, 의지 2, 관찰 3, 지식 1, 영향 1
장비: 강철 검 (근력+1), 가죽 갑옷 (피해-1)
상태이상: 없음

--- 턴 1 ---

[1단계: 공포 판정]
"어둠 속에서 괴물의 형체가 드러난다..."

의지력 2 → D6×2 굴림: [4, 6] → 성공 1개
그림자 괴물 공포값: 4

정신력 피해: 4 - 1 = 3
→ 정신력: 6 → 3
"당신의 마음은 그 존재의 광기에 노출되었다!"

[2단계: 전투 판정]
"공포를 억제하고 검을 든다!"

근력 3 + 검 +1 → D6×4 굴림: [2, 5, 5, 3] → 성공 2개
그림자 괴물 전투값: 3

플레이어 피해: 3 - 2 = 1 → 갑옷 -1 = 0
그림자 괴물 피해: 2 → 체력 8 → 6

"검은 그림자를 찔렀지만 공기를 찌르는 듯했다!"

[적 반응]
"그림자가 당신에게 쏟아진다!"
자동 피해 1 (회피 불가) → 체력 8 → 7

--- 턴 2 ---

[선택지]
- [공격] 근력 판정
- [방어 태세] 다음 피해 -2, 반격 가능
```
