# 던전 탐사 시스템 (Eldritch Horror 기반)

> **재사용 가능한 모듈 시스템**. Eldritch Horror의 주사위 풀, 이중 전투, 뒷면 카드 상태이상, 징조(Reckoning) 시스템을 탑재한 턴제 RPG.

---

## 1. 핵심 개념

### 던전 = 디지털 어드벤처북 + 위협이 도사리는 공간
- 각 던전은 여러 "방(Room)"으로 구성된 텍스트 기반 탐사 공간
- **입구에서 자유롭게 탐사**하다가 **언제든 귀환 가능**
- 모든 전투와 위협은 **하이 리스크 하이 리턴**: 피할 수 없는 대가가 따름

### Eldritch Horror 핵심 시스템 이식
```
1. 주사위 풀 판정: 능력치만큼 D6 굴림, 5-6이 성공
2. 이중 전투: 정신력 판정(의지) → 체력 판정(근력) 순서로 진행
3. 뒷면 카드 상태이상: 언제 터질지 모르는 시한폭탄
4. 징조(Reckoning): 특정 타이밍에 상태이상 뒷면 확인
5. 장비는 "주사위 추가" 형태로만 작용 (스탯 뻥튀기 없음)
```

## 2. 핵심 능력치 및 판정 시스템

### 2.1 5대 능력치

```yaml
"지식" (Lore):
  용도: 고대 유물 판정, 마법 해독, 비밀 문서, 특수 적 약점 파악
  전투: 마법 사용, 특수 아이템 활용
  
"영향력" (Influence):
  용도: NPC 대화, 협상, 거짓말, 동료 설득
  전투: 적과 대화/협상, 패배시 구명 타협
  
"관찰력" (Observation):
  용도: 함정 발견, 숨겨진 문, 적 정보 파악, 흔적 추적
  전투: 선제공격 기회, 적 패턴 예측
  
"근력" (Strength):
  용도: 물리적 장애물, 무거운 문, 도구 사용
  전투: 물리 공격, 방어, 도주
  
"의지력" (Will):
  용도: 공포 저항, 저주 버티기, 유혹 거부
  전투: 공포 판정(필수), 정신 공격 저항
```

### 2.2 주사위 풀 판정 (Dice Pool Test)

```
판정 절차:
1. 요구 능력치 확인 (예: 함정 회피는 "관찰력")
2. 해당 능력치 수치만큼 D6 굴림
3. 5 또는 6이 나오면 "성공(Success)"
4. 성공 개수에 따라 결과 결정

예시:
관찰력 3인 플레이어가 함정을 발견하려 판정
→ D6×3 굴림: [2, 5, 6] → 성공 2개
→ 결과: 함정 발견 + 추가 단서 획득
```

### 2.3 결과 해석 (Eldritch Horror 스타일)

```yaml
성공 개수:
  0개 (실패):
    - 기본 실패 효과 발생
    - 추가 대가 (상태이상, 피해, 적 접근 등)
    
  1개 (부분 성공):
    - 목표 달성하지만 대가 동반
    - 예: "함정은 피했지만 발소리가 들렸다" (적 접근)
    
  2개+ (성공):
    - 목표 완전 달성
    - 보너스 추가 가능
    
  4개+ (대성공):
    - 완벽한 성공
    - 의외의 이득 (희귀 아이템, 숨겨진 정보 등)

난이도 목표치 (대체 판정시):
  쉬움: 1 성공
  보통: 2 성공
  어려움: 3 성공
  극악: 4 성공
```

### 2.4 보정 요소

```yaml
장비 효과 (모두 "+주사위" 형태):
  "예리한 단검": 전투 근력 판정 +1 주사위
  "발광석": 어두운 곳 관찰력 판정 +2 주사위
  "방패": 전투에서 피해 시 의지력 판정 +1 주사위 (공포 저항)
  "고대의 반지": 지식 판정 +1 주사위, 마법 사용시 추가 주사위 1개

상태이상 효과:
  "부상": 근력 판정 -1 주사위
  "공포": 의지력 판정 -1 주사위 (최소 1개 보장)
  "집중": 다음 판정 +2 주사위 (1회용)

환경 효과:
  "어두움": 관찰력 -1 주사위, 근력 -1 주사위
  "좁은 공간": 근력 판정 -1 주사위, 회피 불가
  "고대의 힘": 지식 판정 +1 주사위, 의지력 -1 주사위
```

## 3. 혁신적인 이중 전투 시스템

> 모든 전투는 **반드시 정신력 판정 → 체력력 판정** 순서로 진행

### 3.1 전투 구조

```yaml
적 조우:
  1단계: 공포 판정 (Horror Check) - 필수
    능력치: 의지력 (Will)
    대상치: 적의 "공포값"
    
    결과 계산:
      플레이어 성공 주사위 수: S
      적 공포값: H
      정신력 피해: max(0, H - S)
      
    예시:
      의지력 3 → D6×3 = [2, 4, 5] → 성공 1개
      고블린 공포값: 2
      정신력 피해: 2 - 1 = 1
      
  2단계: 전투 판정 (Combat Check)
    능력치: 근력 (Strength)
    대상치: 적의 "전투값"
    
    결과 계산:
      플레이어 성공 주사위 수: S
      적 전투값: C
      
      플레이어 체력 피해: max(0, C - S)
      적 체력 피해: S
      
    예시:
      근력 3, "예리한 단검" (+1주사위) → D6×4
      결과: [1, 3, 5, 6] → 성공 2개
      고블린 전투값: 3
      
      플레이어 피해: 3 - 2 = 1
      고블린 피해: 2 (고블린 HP 3이면 1 남음)

  3단계: 전투 지속/종료 결정
    - 적 체력 0: 전투 종료, 승리
    - 적 생존: 다음 턴 (플레이어 선택 → 적 반응 순으로 진행)
    - 플레이어 도주: 도주 판정 (관찰력 또는 근력)
```

### 3.2 적 스펙 예시

```yaml
고블린 정찰병:
  체력: 3
  공포값: 1 (약함)
  전투값: 2
  특성: "겁쟁이" - 피해 1 받으면 도주 시도
  
늑대:
  체력: 5
  공포값: 2
  전투값: 3
  특성: "무리" - 2체 이상시 공포값+1, 전투값+1
  
그림자 괴물:
  체력: 8
  공포값: 4 (강함)
  전투값: 3
  특성: "공포유발" - 공포 판정 실패시 "공포" 상태이상 추가
  
랫맨 군주 (보스):
  페이즈: 3
  페이즈1:
    체력: 10
    공포값: 3
    전투값: 4
    특성: "경계" - 첫 전투 판정 플레이어 -1 주사위
  페이즈2:
    체력: 8
    공포값: 4
    전투값: 5
    특성: "격분" - 매 턴 플레이어 피해 +1
  페이즈3:
    체력: 5
    공포값: 5
    전투값: 6
    특성: "최후의 발악" - 1HP시 즉사 공격 1회
```

### 3.3 전투 예시 (풀 세션)

```
=== 그림자 괴물과 조우 ===

[플레이어 상태]
체력: 8/10 | 정신력: 6/8
능력치: 근력 3, 의지 2, 관찰 3
장비: 녹슨 검 (전투 +0), 가죽 조끼 (피해 -1)
상태이상: 없음

[1단계: 공포 판정]
"어둠 속에서 괴물의 형체가 드러난다. 그 존재를 보는 순간, 
당신의 이성이 경고를 울린다."

의지력 2 → D6×2 굴림: [3, 6] → 성공 1개
그림자 괴물 공포값: 4

정신력 피해: 4 - 1 = 3
"당신의 마음은 그 존재의 광기에 노출되었다."
→ 정신력: 6 → 3

[2단계: 전투 판정]
"공포를 억제하고 무기를 든다."

선택지:
- [공격] 근력 판정
- [도주] 관찰력 판정
- [마법 아이템] 지식 판정
→ [공격] 선택

근력 3 + 녹슨 검 (+0) → D6×3
굴림: [2, 5, 5] → 성공 2개
그림자 괴물 전투값: 3

플레이어 피해: 3 - 2 = 1 → 가죽 조끼 (-1) = 0
그림자 괴물 피해: 2 → 체력 8 → 6

"검은 그림자를 찔렀지만, 마치 공기를 찌르는 듯했다. 
괴물은 살아남았다."

[3단계: 적 반응]
그림자 괴물 턴:
"그림자가 당신에게 쏟아진다!"
(자동 피해 1, 회피 불가)
→ 체력: 8 → 7

[다음 턴 선택지]
- [공격] (2턴째: 근력 판정)
- [방어 태세] 다음 피해 -2, 반격 기회
- [집중] 다음 판정 +2 주사위
- [도주] 관찰력 판정
```

### 3.4 도주 시스템

```yaml
도주 판정:
  능력치: 관찰력 (Observation) 또는 근력 (Strength)
  상황에 따라 선택 가능
  
결과:
  실패 (0 성공):
    도주 실패, 적 반격 (자동 피해)
    "발이 묶였다!"
    
  부분 성공 (1 성공):
    도주 성공하지만 대가 발생
    - "피로" 상태이상 획득
    - 또는 아이템 1개 강제 소모/분실
    "간신히 도망쳤지만, 도중에 가방을 떨어뜨렸다."
    
  성공 (2+ 성공):
    깔끔한 도주
    "어둠 속으로 사라졌다."
```

## 4. 뒷면이 숨겨진 상태이상 시스템 (Condition Cards)

> 모든 상태이상은 **뒷면이 가려진 카드** 형태. 징조(Reckoning) 시 뒷면 확인.

### 4.1 상태이상 카드 구조

```yaml
상태이상 카드:
  ID: "상처_1"
  이름: "깊은 상처"
  
  앞면 (Front):
    효과:
      - 근력 판정 -1 주사위
      - 전투 공포값 +1 (고통으로 인한 공포)
    설명: "크게 다쳤다. 움직일 때마다 아프다."
    
  뒷면 (Back) - 징조 발생시 확인:
    징조 조건: "징조 단계" 또는 특정 이벤트
    
    효과:
      근력 판정:
        성공 2개+: "부상이 아물기 시작했다" → 상태이상 제거
        성공 1개: "상태 유지, 다음 징조까지 지속"
        성공 0개: "감염되었다" → 체력 -3, "맹독" 상태이상 추가
        
    설명: "상처에서 고약한 냄새가 난다..."
```

### 4.2 징조(Reckoning) 시스템

```yaml
징조 발생 타이밍:
  1. 던전 탐사 중 "징조 이벤트" 발생
  2. 특정 행동 후 (마법 사용, 저주받은 아이템 사용 등)
  3. 던전 클리어 후 귀환시
  4. 보스 전투 시작시

징조 진행:
  1. 플레이어가 보유한 모든 상태이상 카드 뒷면 확인
  2. 각 카드의 뒷면 지시대로 판정 진행
  3. 결과 적용 (회복/악화/유지)

징조 UI:
  [징조 단계!]
  당신의 상태이상이 시험받습니다...
  
  [상처] 뒷면 확인 → 근력 판정
  [공포] 뒷면 확인 → 의지력 판정
  
  결과 적용 중...
```

### 4.3 상태이상 목록

```yaml
# 물리적 상태이상

상처 (Wound):
  앞면:
    - 근력 -1 주사위
    - 전투시 공포값 +1
  뒷면:
    - 근력 판정 (2성공 필요)
    - 성공: 회복
    - 실패: 체력 -3, "맹독" 추가

출혈 (Bleeding):
  앞면:
    - 매 턴 시작시 체력 -1
    - 근력 -1 주사위
  뒷면:
    - 의지력 판정 (1성공 필요)
    - 성공: 출혈 정지
    - 실패: 체력 -2, "쇼크" 추가

골절 (Fracture):
  앞면:
    - 근력 -2 주사위
    - 이동시 추가 턴 소모
  뒷면:
    - 근력 판정 (3성공 필요)
    - 성공: "골절" → "상처"로 경감
    - 실패: "불구"로 악화

맹독 (Severe Poison):
  앞면:
    - 매 턴 체력 -2
    - 근력 -1 주사위
  뒷면:
    - 지식 판정 (2성공 필요)
    - 성공: 해독
    - 실패: 체력 -5, "마비" 추가

# 정신적 상태이상

공포 (Terror):
  앞면:
    - 의지력 -1 주사위
    - 공포 판정시 공포값 +1
  뒷면:
    - 의지력 판정 (2성공 필요)
    - 성공: 회복
    - 실패: "공황"으로 악화

공황 (Panic):
  앞면:
    - 전투 선택지 1개 랜덤 제거
    - 의지력 -2 주사위
  뒷면:
    - 의지력 판정 (3성공 필요)
    - 성공: "공포"로 경감
    - 실패: 도주 강제, 아이템 1개 분실

집착 (Obsession):
  앞면:
    - 특정 행동 강제 (저주받은 아이템 사용 등)
    - 영향력 -1 주사위
  뒷면:
    - 의지력 판정 (2성공 필요)
    - 성공: 회복
    - 실패: 정신력 -3, 행동 강제 실행

죄책감 (Guilt):
  앞면:
    - 의지력 -1 주사위
    - 휴식 불가
  뒷면:
    - 영향력 판정 (2성공 필요)
    - 성공: 회복
    - 실패: "우울" 추가

# 특수 상태이상

저주 (Curse):
  앞면:
    - 매 탐사 시작시 1D6: 1이면 랜덤 악 이벤트
    - 모든 판정 -1 주사위
  뒷면:
    - 지식 판정 (3성공 필요)
    - 성공: 저주 해제
    - 실패: 최대 정신력 -1 (영구적)

계약 (Pact):
  앞면:
    - 특정 혜택 (무료 아이템, 능력치 +1 등)
  뒷면:
    - 의지력 판정 (4성공 필요 - 극악)
    - 성공: 계약 파기, 혜택 회수
    - 실패: 체력 또는 정신력 절반 즉시 손실, 또는 사망

표식 (Mark):
  앞면:
    - 특정 적들이 항상 플레이어 우선 타겟
    - 공포값 +1
  뒷면:
    - 특정 이벤트 발생 (추격, 암살 시도 등)
    - 또는 "저주"로 악화
```

### 4.4 상태이상 연쇄 (Eldritch Horror 스타일)

```yaml
자동 악화:
  "상처" ×2 보유 + 징조 실패 → "맹독"
  "공포" ×2 보유 + 징조 실패 → "공황"
  "출혈" + 징조 실패 → "쇼크" → 전투 불능
  
치명적 연쇄:
  "저주" + "집착" 보유시 징조 발생
    → 자동 실패
    → 특수 이벤트 "고대 존재의 주시" 발생

회복 연쇄:
  "공황" 징조 성공 → "공포"로 경감
  "공포" 징조 성공 → 완전 회복
```

      텍스트: "간신히 도망쳤다."

## 6. 장비 및 마법 시스템

### 6.1 장비 원칙

```yaml
장비는 "+주사위" 형태만 가능:
  "강철 검": 전투 근력 판정 +1 주사위
  "은빛 단검": 유령과 전투시 근력 +2 주사위
  "방패": 피해 감소 1 (고정값)
  "고대의 반지": 지식 판정 +1 주사위
  "신속의 장화": 도주 관찰력/근력 판정 +1 주사위
  "발광석": 어두운 곳 모든 판정 +1 주사위

장비는 영구적 능력치 상승이 아님:
  - 특정 상황에서만 발동
  - 파손 가능 (전투 실패시)
```

### 6.2 마법 시스템

```yaml
마법 사용:
  1. 지식 판정 (발동 판정)
  2. 성공시 효과 발생
  3. 마법 카드 뒷면 확인 (징조 효과)

마법 예시: "화염의 손길"
  발동: 지식 판정 2성공 필요
  효과: 적 체력 -4 (즉시)
  
  뒷면 (징조 발생시):
    지식 판정:
      성공 2+: 아무 일 없음
      성공 1: 정신력 -1 (마법의 대가)
      실패: "과열" - 체력 -2, 아이템 1개 파손
      
마법 예시: "정신 보호막"
  발동: 지식 판정 1성공
  효과: 다음 공포 판정시 의지력 +2 주사위
  
  뒷면:
    의지력 판정:
      성공: 아무 일 없음
      실패: "정신 쇠약" - 다음 턴 행동 불능
```

## 7. 귀환 시스템

### 7.1 귀환 방법

```yaml
1. 입구로 돌아가기:
   - 지금까지 온 길을 역순으로 되돌아감
   - 각 방마다 "귀환 이벤트" 발생 (쉬운 난이도)
   - 시간 소모: 탐사 시간의 1/3
   
2. 귀환 아이템:
   - "귀환석": 즉시 마을 복귀
   - "연막탄": 전투에서 즉시 도주 + 귀환
   
3. 전투 불능:
   - 체력 0 또는 정신력 0
   - "검은 그림자"가 구출 (페널티 있음)
```

### 7.2 귀환 결과

```yaml
성공적 귀환:
  - 보유 아이템: 100% 유지
  - 획득 정보: 100% 유지
  - 상태이상: 유지 (여관에서 치료 필요)

비상 귀환 (전투 불능):
  - 아이템: 50% 무작위 분실
  - 돈: 30% 분실
  - 상태이상: "공포" 추가
  - 징조: 즉시 발생 (모든 상태이상 뒷면 확인)
```
