# 컨셉 5: Idle 농장 운영 + 연애 게임

## 핵심 컨셉
방침만 정해주면 되는 idle 타입의 농장 운영 게임. 직접 노가다가 아닌 스케줄 지정으로 진행되며, 지정된 외출을 통해 NPC와 만나 연애를 할 수 있음.

## 게임플레이
- 어떤 작물을 심고 어떤 활동을 할지 스케줄만 지정
- 시간이 지나면 자동으로 수확/진행
- 외출 스케줄로 마을 내 NPC와 만남 및 연애 이벤트
- 비주얼 노벨 형식의 대화 + 간단한 UI

## 비교 대상
Stardew Valley (경량화) / Harvest Moon (스케줄 중심) / Coffee Talk

## 장점
- 비주얼 노벨 엔진과 가장 잘 맞는 단순한 구조
- idle 방식이라 시스템 프로그래밍이 거의 불필요함
- 연애 이벤트 중심으로 시나리오 라이터의 강점 극대화
- 2개월 내 개발에 가장 적합한 규모
- "귀여운 여자애들"과의 연애가 핵심 콘텐츠가 됨

## 단점
- idle이라는 특성상 플레이어의 직접 조작감이 약함
- Papers, Please만큼의 참신함은 없을 수 있음
- 농장이라는 컨셉이 흥미롭지 않게 느껴질 수 있음 (기획에 따라 차이 큼)
