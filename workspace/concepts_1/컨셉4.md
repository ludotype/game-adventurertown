# 컨셉 4: 프린세스 메이커 클론 (UI 경량화)

## 핵심 컨셉
어려진 여마왕/드래곤을 키워내는 프린세스 메이커 스타일 육성 시뮬레이션. UI를 최소화하고 스토리와 메인 캐릭터에 집중.

## 게임플레이
- 메인 캐릭터(여마왕/드래곤)에게 매월/매주 스케줄 지정
- 교육, 아르바이트, 휴식, 외출 등 선택에 따른 능력치 변화
- 특정 능력치/이벤트 충족 시 분기 엔딩
- NPC 수를 줄여 전체 규모를 작게 구성

## 엔딩 예시
- 음악가 엔딩
- 왕국 수호 드래곤 엔딩
- 마왕 엔딩
- 연애 엔딩 등

## 비교 대상
Princess Maker 시리즈 / Long Live The Queen / Cute Knight Kingdom

## 장점
- 비주얼 노벨 엔진과 완벽하게 호환됨 (스케줄 선택지 + 이벤트)
- 메인 캐릭터 하나에 집중하므로 아트 리소스 집중 가능
- "귀여운 여자애"가 메인 캐릭터로 자연스럽게 중심에 서 있음
- 시나리오 라이터로서의 강점을 최대한 발휘 가능
- 2개월 내 혼자 개발하기에 현실적인 규모

## 단점
- 프린세스 메이커는 이미 잘 알려진 장르라 참신함이 다소 부족할 수 있음
- UI가 단순해도 육성 시뮬레이션의 핵심 로직(능력치, 이벤트 분기)은 필요함
- 반복 턴제 gameplay가 지루하게 느껴질 수 있음
