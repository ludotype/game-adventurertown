# 컨셉 5 평가: Idle 농장 운영 + 연애 게임

## 평가 기준
사용자의 조건: Papers Please급 참신함, 귀여운 여자애들+재미있는 대사, 아티스트+시나리오 라이터 혼자 2개월 완성, 비주얼 노벨 엔진 연계, 최소 프로그래밍

## 1. 참신함 / 인터랙티브함: C
- 농장 게임은 Stardew Valley, Harvest Moon 등 수많은 작품이 있는 흔한 장르
- Idle 타입이라 플레이어의 직접 조작감이 약함
- Papers Please만큼의 참신함이나 인터랙티브함은 기대하기 어려움
- Coffee Talk 같은 방식으로 차별화할 수는 있으나, 기반 컨셉(농장)이 너무 평범함

## 2. 귀여운 여자애들 / 재미있는 대사 적합성: A
- 마을 내 NPC와의 연애가 핵심 콘텐츠이므로 "귀여운 여자애들"이 자연스럽게 등장함
- 연애 이벤트 중심으로 시나리오 라이터의 강점을 극대화할 수 있음
- 외출 스케줄마다 다른 NPC를 만나는 구조는 대사량이 풍부해짐

## 3. 2개월 혼자 완성 가능성: A
- Idle 방식이라 시스템 프로그래밍이 거의 불필요함
- 스케줄 선택 → 시간 경과 → 이벤트 발생이라는 단순한 루프
- 비주얼 노벨 엔진으로 구현하기에 가장 쉬운 구조
- 2개월이면 충분히 완성 가능하며, 오히려 1개월이면 기본 뼈대가 나올 수 있는 규모

## 4. 비주얼 노벨 엔진 연계성: A
- 스케줄 선택 + 시간 경과 + 연애 이벤트는 VN 엔진의 기본 기능으로 충분함
- 추가 UI가 거의 필요 없음

## 5. 최소 프로그래밍 필요도: 매우 낮음
- 시간 경과와 스케줄-이벤트 매핑 외에는 거의 프로그래밍이 필요 없음

## 종합 평가
- **구현 난이도는 5개 중에서 가장 낮고, 완성 가능성은 가장 높음**
- 귀여운 여자애들과 연애 이벤트 중심이라 사용자의 강점과도 잘 맞음
- 그러나 **"Papers Please 같이 참신한"**이라는 사용자의 원래 요구사항과는 거리가 있음
- 농장이라는 컨셉이 너무 평범하여 플레이어의 첫인상이 "또 다른 농장 게임?"이 될 위험이 큼
- **보류**: 구현은 쉬우나 참신함이 부족함. 단, 농장 대신 다른 설정(예: 카페 운영, 서점 운영, 마법 재료 상점 등)으로 바꾸면 가치가 크게 상승할 수 있음
