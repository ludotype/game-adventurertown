## 문서 작성 규칙

1. **파일명**: `snake_case.md` 또는 `PascalCase.md`
2. **포맷**: Markdown (.md)
3. **언어**: 한국어 (기본), 코드/기술 용어는 영어 병기
4. **연결**: 각 문서 하단에 **관련 문서** 섹션을 반드시 포함
