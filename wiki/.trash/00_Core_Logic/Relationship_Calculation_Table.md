# 관계도 계산 테이블 (Relationship Calculation Table)

## 1. 개요

캐릭터 간의 호감도는 **성격 태그(Personality Tags)**의 상호작용에 따라 동적으로 계산됩니다. 모든 관계 변화는 시스템 로그에 기록되며, 뉴스 티커와 이벤트 트리거의 기반이 됩니다.

---

## 2. 성격 태그 카테고리

### 2.1. 기본 성향 태그 (Core Traits)
| 태그 | 설명 | 반대 태그 |
|------|------|-----------|
| `탐욕스러운(Greedy)` | 전리품 우선, 이익 추구 | `관대한(Generous)` |
| `금사빠(Romantic)` | 쉽게 호감 형성, 연애 중심 | `냉혈한(Cold-hearted)` |
| `냉혈한(Cold-hearted)` | 감정 변화 둔감, 효율 우선 | `다정한(Warm)` |
| `질투심 강한(Jealous)` | 타인의 성공/관계에 민감 | `대범한(Magnanimous)` |
| `의리파(Loyal)` | 동료 우선, 배신 저항 | `기회주의자(Opportunist)` |
| `신중한(Cautious)` | 위험 회피, 철저한 준비 | `무모한(Reckless)` |
| `허세부리는(Braggart)` | 자신 과시, 주목 욕구 | `겸손한(Humble)` |
| `다정한(Warm)` | 타인에게 친근함 표현 | `냉혈한(Cold-hearted)` |

### 2.2. 특수 성향 태그 (Special Traits)
| 태그 | 설명 | 효과 |
|------|------|------|
| `불운한(Unlucky)` | 사고 발생 확률 증가 | 전투/탐험 중 부정 이벤트 +15% |
| `재능있는(Talented)` | 성장 속도 증가 | 경험치 획득 +20% |
| `집착하는(Obsessive)` | 한 목표에 몰입 | 한 관계의 호감도 변화 2배 |
| `방랑자(Wanderer)` | 정착을 싫어함 | 길드 충성도 감소 속도 2배 |

## 3. 태그 조합별 호감도 계산식

### 3.1. 기본 상호작용 계수
```
호감도 변화 = (기본값 × 태그 계수 A × 태그 계수 B × 상황 계수) + 랜덤 변동(-5 ~ +5)
```

### 3.2. 태그 쌍 상호작용 테이블

#### 긍정적 상호작용 (+)
| 태그 A | 태그 B | 계수 | 설명 |
|--------|--------|------|------|
| `금사빠` | `금사빠` | ×1.5 | 서로 빠르게 호감 형성 |
| `의리파` | `의리파` | ×2.0 | 강한 유대감, 전투 시 시너지 |
| `의리파` | `신중한` | ×1.3 | 서로를 보완하는 관계 |
| `다정한` | `냉혈한` | ×1.2 | 상극이지만 서로 끌림 |
| `관대한` | `탐욕스러운` | ×0.8 | 약간의 긴장감 있음 |
| `겸손한` | `허세부리는` | ×1.0 | 균형 잡힌 관계 |

#### 부정적 상호작용 (-)
| 태그 A | 태그 B | 계수 | 설명 |
|--------|--------|------|------|
| `질투심 강한` | `재능있는` | ×-1.5 | 재능 있는 자를 질투 |
| `질투심 강한` | `허세부리는` | ×-1.2 | 자랑에 분노 |
| `냉혈한` | `금사빠` | ×-1.0 | 감정적 접근에 거부감 |
| `탐욕스러운` | `관대한` | ×-0.8 | 전리품 배분 시 충돌 |
| `무모한` | `신중한` | ×-0.6 | 행동 방식 충돌 |
| `의리파` | `기회주의자` | ×-2.0 | 극도의 적대 |

## 4. 상황별 호감도 변동값

### 4.1. 전투 상황
| 이벤트 | 기본값 | 특수 태그 효과 |
|--------|--------|----------------|
| 함께 승리 | +10 | `의리파`: +15, `냉혈한`: +5 |
| 위기 구출 | +20 | `금사빠`: +25 (감동), `집착하는`: +30 |
| 치유 받음 | +8 | `다정한`: +12 |
| 전리품 다툼 | -15 | `탐욕스러운`: -5 (덜 불만), `관대한`: -20 |
| 동료가 쓰러짐 | -5 | `의리파`: -10 (죄책감), `냉혈한`: 0 |

### 4.2. 사회적 상황
| 이벤트 | 기본값 | 특수 태그 효과 |
|--------|--------|----------------|
| 주점 대화 | +5 | `허세부리는`: +8 (주목받음), `신중한`: +3 |
| 선물 교환 | +15 | `탐욕스러운`: +20, `금사빠`: +18 |
| 비밀 공유 | +12 | `신중한`: +18 (신뢰 중시) |
| 말다툼 | -8 | `질투심 강한`: -15, `허세부리는`: -12 |
| 배신 발각 | -50 | `의리파`: -100, 기회주의자: -20 |

### 4.3. 던전 탐험 상황
| 이벤트 | 기본값 | 특수 태그 효과 |
|--------|--------|----------------|
| 함께 탐험 | +3/턴 | `무모한`: +5, `신중한`: +1 |
| 보물 발견 | +8 | `탐욕스러운`: 보물 독독 시 -30 |
| 함정 탈출 | +10 | `신중한`: +15 (경험 공유) |
| 길 잃음 | -5 | `방랑자`: +5 (익숙함) |
| 야영 | +6 | `다정한`: +10, `냉혈한`: +2 |

## 5. 관계 상태 임계값

```
호감도 범위 | 상태 | 효과
------------|------|------
-100 ~ -51 | 적대 | 전투 시 서로 방해, 협동 불가
-50 ~ -11 | 불편 | 함께 파티 시 전체 능력 -10%
-10 ~ +9 | 중립 | 기본 상태
+10 ~ +29 | 친구 | 전투 시 서로 버프 +5%
+30 ~ +49 | 친밀 | 함께 시 스트레스 감소
+50 ~ +79 | 연인/절친 | 특수 스킬 잠금 해제
+80 ~ +100 | 영혼의 동반자 | 콤보 기술, 죽음 시 복수 버프
```

## 6. 이머전트 이벤트 트리거

### 6.1. 부정적 이벤트
| 조건 | 이벤트 | 결과 |
|------|--------|------|
| 동일 NPC에게 호감도 +30 이상 × 2명 | 치정 싸움 | 결투 발생, 패자 길드 탈퇴 또는 사망 |
| 호감도 -50 이하 + 전리품 분배 | 배신 | 보물 도주, 길드원 -1 |
| 질투심 + 재능있음 관찰 | 음해 | 파티 내 루머 확산 |
| `무모한` × 3 파티 | 브레인 브리지 | 함정 폭발, 전원 사망 확률 |

### 6.2. 긍정적 이벤트
| 조건 | 이벤트 | 결과 |
|------|--------|------|
| `의리파`가 위기에서 `냉혈한` 구출 | 얼음 녹이기 | 두 캐릭터 고유 관계 스킬 해제 |
| 호감도 +80 달성 | 영혼의 맹세 | 콤보 스킬 습득 |
| `신중한`이 `무모한`을 3회 구출 | 멘토십 | `무모한`이 신중 특성 부분 획득 |

## 7. 의사결정 플로우

```
[이벤트 발생]
    ↓
[태그 추출: A, B 캐릭터]
    ↓
[상황 분류: 전투/사회/탐험]
    ↓
[기본값 + 태그 계수 + 상황 계수 계산]
    ↓
[호감도 업데이트]
    ↓
[임계값 체크 → 관계 상태 변화?]
    ↓
[이머전트 이벤트 트리거 체크]
    ↓
[뉴스 티커 생성 로그 기록]
```

## 8. 디버그/밸런스 도구

### 8.1. 테스트 시나리오
```yaml
scenario_1:
  name: "의리파 듀오"
  char_a: { tags: [의리파, 신중한] }
  char_b: { tags: [의리파, 무모한] }
  events: [전투승리×3, 위기구출×1]
  expected_final: +65 (친밀 상태)

scenario_2:
  name: "질투의 덫"
  char_a: { tags: [질투심강한, 재능있는] }
  char_b: { tags: [허세부리는, 재능있는] }
  char_c: { tags: [금사빠] }  # B가 C에게 관심
  events: [C가 B에게 선물, A가 관찰]
  expected_final: A→B: -35 (불편 상태)
```
