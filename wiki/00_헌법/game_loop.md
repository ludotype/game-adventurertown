# 게임 루프 & 일일 사이클

> 1인칭 모험가 연애 어드벤처의 일상 루프 구조

---

## 1. 핵심 루프 개요

```
[마을 장소 이동/행동] → [시간 소비] → [NPC 조우/대화] → [관계 변화] → [다음 행동 or 취침] → [새로운 아침]
```

플레이어는 모험가로서 하루를 보냅니다. 장소를 이동하고, 행동을 선택하고, NPC와 대화하며 시간을 보냅니다. 매일의 작은 선택이 누적되어 루이제와의 관계가 변화합니다.

---

## 2. 일일 사이클 (Daily Cycle)

### 2.1 시간 흐름

```
새벽(05:00-07:00) → 아침(07:00-12:00) → 점심(12:00-13:00)
→ 오후(13:00-18:00) → 저녁(18:00-22:00) → 밤(22:00-24:00) → 심야(00:00-05:00)
```

- 24시간제. 행동마다 시간 단위 소비 (기본 1단위 = 30분)
- 취침 시 다음 날 아침으로 이동
- NPC는 시간대별로 다른 장소에 등장

### 2.2 플레이어 행동

| 행동 유형 | 설명 | 시간 소비 |
|-----------|------|----------|
| **장소 이동** | 인접 장소로 이동 | 이동 시 소비 |
| **장소 행동** | 둘러보기, 쉬기, 대기 등 | 행동별 소비 |
| **NPC 대화** | 등장한 NPC와 대화 | 대화 내용에 따라 |
| **NPC 상호작용** | talk, touch 등 | 상호작용별 |
| **던전 출발** | 던전 탐험 세션 시작 | 탐험 종료까지 |
| **취침** | 다음 날 아침까지 스킵 | 하루 종료 |

---

## 3. 마을 생활 페이즈 (Town Life)

### 3.1 아침 루틴

- 여관방에서 시작
- 복도 → 로비로 이동
- 루이제와 조우 가능 (아침 업무 중)

### 3.2 낮 활동

- 마을 거리/광장 이동
- 상점 방문 (무기상, 꽃집 등)
- NPC 조우 (시간대별 스케줄)
- 던전 출발 선택 가능

### 3.3 저녁/밤

- 주점에서 NPC들과 시간 보내기
- 루이제가 저녁/밤에 주점에 등장
- 대화/상호작용으로 호감도 변화

### 3.4 취침

- 여관방에서 잠자기
- 다음 날 아침 07:00으로 이동
- 일일 플래그 리셋 (rested_today 등)

---

## 4. 던전 탐험 세션 (Adventure Session)

> 텍스트 어드벤처 북 스타일. VN 대화 창에서 진행.

### 4.1 탐험 흐름

```
[던전 입장] → [이벤트 모듈 진행] → [선택지 분기] → [수치 변화(HP/정신/탐색도)] → [귀환 판단] → [마을 복귀]
```

### 4.2 탐험 메카닉

- 각 이벤트 = 텍스트 묘사 + 선택지 + 간단한 수치 변화
- HP, 정신, 탐색도 관리
- "언제 돌아갈지"가 핵심 전략
- 전투는 구색만 맞추는 수준. 플레이어는 영웅이 아님

### 4.3 귀환 후

- 탐험 결과에 따른 보상/상태 변화
- 귀환 시각이 루이제와의 관계에 영향 가능
- 예: 늦은 귀환 → 루이제 걱정 이벤트

---

## 5. 관계 진행 루프 (Relationship Loop)

### 5.1 호감도 누적

```
[대화/행동] → [npc.luise.affection 변화] → [조건부 이벤트 해금] → [새로운 대화/상호작용 가능]
```

### 5.2 단계별 경험

| 단계 | affection | 가능한 상호작용 |
|------|-----------|---------------|
| 무관심 | 0-2 | 기본 대화만, touch 시 놀람 |
| 관심 | 3-5 | 대화 분기 증가, touch 반응 변화 |
| 친밀 | 6+ | 특별 대화, 함께 시간 보내기 |

### 5.3 일상이 관계에 미치는 영향

- 매일 펍에 들러 말 거르기 → 누적 호감도 상승
- 루이제의 의뢰 우선 처리 → 호감도 가속
- 늦은 귀환 → 걱정 이벤트 (양날의 검)
- 다른 NPC와의 교류 → 간접 영향

## 6. 세션 시간 분배

| 활동 | 예상 시간 | 비율 |
|------|----------|------|
| 마을 이동/행동 | 3-5분 | 35% |
| NPC 대화/상호작용 | 3-5분 | 35% |
| 던전 탐험 | 0-5분 (선택적) | 20% |
| 결산/정리 | 1분 | 10% |
| **총 1일** | **7-16분** | 100% |

## 7. 참고 시스템 비교

| 요소 | 페르소나 | 하베스트 문 | Guild Master |
|------|----------|------------|-------------|
| **시간 관리** | 1일 단위 스케줄 | 1일 단위 작업 | 1일 행동 + 시간 소비 |
| **관계** | 커뮤니티 링크 | 결혼 후보 | 단일 히로인 단계별 |
| **탐험** | 메멘타토 | 광산 | 텍스트 어드벤처 북 |
| **엔딩** | 루트 분기 | 결혼 이벤트 | 엔딩 없음, 상태 유지 |
| **핵심 동기** | 관계 완성 | 농장+가족 | 루이제와 친해지기 |
