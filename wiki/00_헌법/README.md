## 7대 설계 원칙 (요약)

1. **Auto-Discovery** — JSON을 폴더에 넣으면 자동 등록. 중앙 인덱스 없음.
2. **ID-First Referencing** — 모든 엔티티는 고유 ID. 파일 경로 직접 참조 금지.
3. **Definition ≠ State** — JSON은 읽기 전용. 런타임 상태는 별도 싱글톤.
4. **Vocabulary, not Table** — 행동/조건/효과를 작은 어휘로 조합.
5. **EventBus 결합** — 시스템 간 직접 호출 금지. 신호로만 소통.
6. **Composable Depth** — 단순/복잡 케이스 모두 같은 스키마.
7. **Fail Fast 검증** — 부팅 시 ID 참조 무결성 검사.
