### Q1. 스탯 세트 선택
**질문**: 판정에 사용할 기본 스탯을 어떤 세트로 확정할까요?

| 선택지 | 구성 |
|---|---|
| **A. 아컴 7** | 근력, 민첩, 지식, 의지력, 행운, 관찰력, 영향력 |
| **B. 심플 5** | 육체(근+민), 정신(지+의), 통찰, 운, 인망 |
| **C. 판정 6** | 근력, 민첩, 지식, 의지력, 행운, 통찰(관찰+영향력 통합) |
| **D. 직접 제안** | 위에 없는 조합을 직접 말씀해 주세요 |

**왜 중요한가**: 모든 판정, 장비, 전투, NPC 대사의 `[근력]` `[관찰력]` 등이 이 스탯에 맞춰져야 합니다. 나중에 바꾸면 수십 곳을 고쳐야 합니다.

**현재 기준**: 문서 전반에 이미 `[근력]`, `[민첩]`, `[지식]`, `[의지력]`, `[행운]`, `[관찰력]`, `[영향력]`이 언급되어 있어 A가 가장 자연스럽습니다.

**답변 예시**: "A", "C로 하고 영향력은 통찰에 넣자", "7개가 너무 많아. 근력 민첩 지식 의지 행운 5개만 쓰자"

### Q2. 파생 수치 (HP / Sanity 등)
**질문**: 스탯이 확정되면, 아래 파생 수치들의 계산 방식을 정의해야 합니다. 원하는 방식을 간단히 답해 주세요.

- **HP (체력)**: 근력 기반? 고정값? (예: "근력x10 + 20", "모든 캐릭터 고정 100")
- **Sanity (정신력)**: 의지력 기반? 고정값?
- **인벤토리 제한**: 근력이 들 수 있는 무게? 고정 슬롯 수?
- **방어도/회피**: 민첩 기반? 장비만?

**왜 중요한가**: 전투 시스템, 장비 시스템, 던전 밸런싱의 기초가 됩니다.

**답변 예시**: "HP=근력x10+20, Sanity=의지력x10+20, 인벤토리는 슬롯 12개 고정"

### Q3. 캐릭터 생성 방식
**질문**: 게임 시작 시 캐릭터를 어떻게 만드나요?

- **Background (배경)**: 사냥꾼/학자/도적/신관/전사 등 프리셋? 자유 분배?
- **스탯 분배**: 포인트 바이? 주사위 굴림? 배경 고정값 + α?
- **시작 장비**: 배경별로 다르게 주나요? 랜덤?
- **특성/단점**: 롤플레이용 텍스트 특성이 있나요? (예: "[밤의 아이] — 야간 조우 시 회피 +1")

**왜 중요한가**: 로그라이트 반복 플레이의 핵심 재미입니다. 배경마다 첫 세션의 플레이 방식이 크게 달라집니다.

**답변 예시**: "프리셋 4종(사냥꾼/학자/도적/신관). 각 배경마다 스탯 고정 분배 + 장비 고정. 특성은 없고 장비/스탯으로 표현"

### Q4. 기본 AP (하루 행동력)
**질문**: 하루 기본 AP를 몇으로 확정할까요? 20 그대로? 조정?

**왜 중요한가**: 전체 게임의 템포를 결정합니다. 20이면 하루에 4~6개 행동. 15면 3~4개. 30면 6~8개.

**현재 기준**: 문서 전반에 이미 "하루 기본 20 AP"가 언급되어 있고, 특수행동들의 비용도 2~5AP로 설계되어 있습니다.

**답변 예시**: "20 유지", "15로 줄이고 긴장감 높이자", "25로 늘려 여유두자"

### Q5. AP 변동 요인
**질문**: 어떤 요인들이 AP에 영향을 주나요? (+/- 몇 AP 또는 퍼센트로 대략적으로)

- **Background**: 배경마다 AP 보너스/페널티? (예: 사냥꾼 +2AP, 신관 -1AP)
- **유물/장비**: 특정 아이템이 AP 상한을 늘려주나요? (예: "시계태엽 심장 — AP +3")
- **상태이상**: `[부상]`, `[숙취]`, `[인지 왜곡]` 등이 AP에 영향?
- **장소 효과**: 선술집에서 숙박하면 다음 날 +1AP?
- **NPC 조력**: 루이제와 함께 밤을 넘기면 다음 날 AP 회복 +α?

**왜 중요한가**: 매 세션마다 AP가 달라지는 "재미있는 불공정"을 설계합니다.

**답변 예시**: "유물/장비만 AP에 영향. 배경은 영향 없음. 상태이상은 AP가 아니라 스탯에 페널티. 장소/NPC 효과는 다음 날 AP 회복 +2 정도"

| **B. 액티브 스킬** | 던전/전투 중 "루이제의 도움 요청"으로 1회용 강력 효과. |
| **C. 동행자 (Follower)** | 실제로 함께 던전 탐사. HP/Sanity/스킬 보유. |

**왜 중요한가**: A가 구현 가장 간단, C가 가장 복잡하지만 RPG 깊이는 최고.

**답변 예시**: "A+B 혼합. 동행자는 나중에 DLC나 업데이트로"

### Q17. 해금 포인트 기준
**질문**: 세션 종료 시 어떤 기준으로 해금 포인트를 부여하나요?

- 위기 해결 개수당?
- 던전 클리어 개수당?
- 생존 일수당?
- 특수 업적 ("한 번도 부상 없이 클리어" 등) 보너스?

**왜 중요한가**: 로그라이트의 "또 한 판" 심리를 만드는 핵심 보상 설계입니다.

**답변 예시**: "위기 해결 1개 = 100P, 던전 클리어 = 50P, 생존 1일 = 10P, 업적 = 200P"

### Q18. 해금 콘텐츠
**질문**: 반복 플레이로 무엇을 해금할 수 있나요?

| 종류 | 예시 |
|---|---|
| 새 배경 | 기본 3종 → 해금으로 8~10종 |
| 시작 유물 | 세션 시작 시 1개 지닌 채 시작 |
| 신화 카드 | 신화 단계에 추가될 새로운 재난/이벤트 |
| 숨겨진 위기 | 5개 메인 외 보너스 Crisis |
| 숨겨진 엔딩 | 진엔딩/진정한 최종 보스 |

**왜 중요한가**: 1인 개발 시 너무 많은 해금은 콘텐츠 폭증을 야기합니다. 우선순위를 정해야 합니다.

**답변 예시**: "배경 + 시작 유물만 1.0에서. 나머지는 업데이트"

### Q19. 메인 음모 구조
**질문**: 5개의 메인 위기(음모)는 어떤 구조를 가지나요?

| 선택지 | 설명 |
|---|---|
| **A. 순차적** | 1번 클리어 → 2번 unlock → ... → 5번이 최종 보스로 연결 |
| **B. 병렬적** | 5개가 동시에 활성화 가능. 플레이어가 순서를 선택 |
| **C. 하이브리드** | 3개는 병렬, 2개는 이전 위기 클리어 후 unlock |

**왜 중요한가**: 병렬이면 플레이어에게 전략적 자유도가 높지만, 서사의 긴장도가 분산될 수 있습니다.

**답변 예시**: "C. 처음 2개는 병렬로 열리고, 3번은 하나 클리어 후, 4~5번은 둘 다 클리어 후"

### Q20. 최종 결전
**질문**: 5개 음모 클리어 후 무엇이 기다리나요?

| 선택지 | 설명 |
|---|---|
| **A. 보스 전투** | The Slumbering Maw와의 직접 전투 |
| **B. 의식/봉인** | 전투가 아닌, 특정 단서 아이템 조합 + 의식 판정 체인 |
| **C. 도시 탈출/희생** | 플레이어가 희생하여 도시를 구함 vs. 도망쳐서 살아남음 |

**왜 중요한가**: 게임의 마지막 인상을 결정합니다. 텍스트 RPG에서는 B가 서사적으로 강력할 수 있습니다.

**답변 예시**: "A+B 혼합. 보스와 전투하되, 최후의 일격은 봉인 의식으로 처리"

| Bad Ending | 미정의 | |
