# 자율 액터 (NPA) 시뮬레이션 시스템 설계서

> 이 문서는 플레이어와 무관하게 고대 도시 나틀락(Natlach)에서 자율적으로 생활하고 던전을 탐사하며 위기에 대응하는 **자율 액터(NPA: Non-Playable Actors)** 시뮬레이션 시스템 사양서입니다.
> **핵심 사양**: 턴제 코스믹 호러 RPG 피벗 사양(`new_direction.md`)을 100% 반영하여 수치적 정신력을 완전 배제하고 **상태 카드(Condition Cards)**와 **스태미나(Stamina)** 메커니즘을 동일하게 도입합니다.

---

## 1. 시스템 개념 및 자율 활동 주기 (Daily Cycle)

플레이어는 절망적인 나틀락(Natlach)를 살아가고 있는 "수많은 모험가 중 한 명"일 뿐입니다. 도시에는 전사, 신관, 도적 등 각자의 목적과 위기 대응 방식을 지닌 자율적인 동료들이 독립적으로 존재합니다.

NPA 시뮬레이션은 매일 자정(Midnight Phase)에 갱신되며, 다음과 같은 고유 주기로 작동합니다.

```
[ 자정 (Midnight Phase) ]
NPA 상태(HP/스태미나/상태 카드) 자가 점검
  │
  ├─ 상태 불량 (HP ≤ 20% 혹은 중상 카드 보유 시) ───────────────────┐
  │  → 마을에 강제 잔류하여 휴식 및 상태 카드 정화 시도                   │
  │                                                                │
  └─ 상태 양호 (HP/스태미나 충족 시)                                     │
     → '기본 던전 출발 확률' 주사위 굴림                                  │
         │                                                         │
         ├─ 실패 (마을 잔류) ──────────────────────────────────────┼─→ [ 마을 생활 스케줄 가동 ]
         │                                                         │   - 시간대별 장소 이동
         └─ 성공 (던전 출발)                                        │   - 플레이어와의 우호적 조우
              │                                                    │   - 마을 휴식 시 스태미나 회복
              ├─ TownNews에 출발 알림 송출                             │
              ├─ 귀환 시간대(아침/오후/저녁/밤) 예약                      │
              │                                                    │
              └─ [ 던전 탐사 결과 격발 (귀환 시각) ]                       │
                 - 주사위 굴림으로 결과 판정(대성공/성공/실패/대실패)       │
                 - 자원(단서 토큰, 아이템) 획득 및 HP/스태미나 손실        │
                 - 결과에 따른 긍정적/부정적 상태 카드 획득                 │
                 - TownNews에 결과 보고 알림 송출                        │
                                                                   │
                                                                   └─→ [ 익일 점검 주기로 연결 ]
```

## 2. 상태 카드(Condition Cards) 및 자원 메커니즘

수치 형태의 정신력(Sanity)은 존재하지 않으며, 플레이어와 완벽하게 동일한 **HP, 스태미나(Stamina), 상태 카드(Condition Cards)** 구조를 NPA에게 동적 매핑합니다.

### 2.1 NPA 고유 상태 카드
NPA가 던전 탐사에서 실패하거나 특수 이벤트를 마주하면 다음과 같은 고유 상태 카드가 부여되며, 대사 및 자율 활동 확률에 직접 개입합니다:

| 상태 카드 (ID) | 성격 | NPA 행동에 미치는 영향 |
|---|---|---|
| **축복받은 (blessed)** | 긍정적 | 던전 탐사 성공률 +15%, 대성공 확률 2배 상승. |
| **탈진한 (exhausted)** | 부정적 | 스태미나가 20 이하로 고착되며, 던전 출발 확률 -30%. |
| **부상당한 (wounded)** | 부정적 | 매 턴 시작 시 HP가 5씩 지속적으로 감소. |
| **편집증 (paranoid)** | 정신 이상 | 플레이어에게 단서 토큰(Clue Tokens) 제공 요청 시 필요 신뢰도가 2배로 상승. |
| **공포에 질린 (panicked)** | 정신 이상 | 던전 출발을 원천 거부하고 주점이나 여관방에서 굳어짐. |
| **환각 (hallucinating)** | 정신 이상 | 대화 시 알 수 없는 헛소리를 지껄임. 자정마다 10% 확률로 던전에 멋대로 출발. |

## 3. 던전 탐사 및 귀환 판정 (Dungeon Run Resolution)

예약된 귀환 시간대(Time Block)가 되면 NPAAutonomyManager가 해당 NPA의 탐사 결과를 난수 주사위(0.0 ~ 1.0)를 굴려 판정합니다.

### 3.1 결과 판정 등급 및 결과 효과

#### A. 대성공 (Critical Success) — 확률 15% (blessed 시 30%)
- **자원 획득**: **단서 토큰 2개** 획득, 전용 유물(Rare Loot) 1개 획득.
- **스탯 변화**: HP 손실 절반 경감, 스태미나 10 소소하게 소모.
- **카드 변화**: `blessed` 상태 카드를 획득하고 기존 부정적 정신 상태 카드들을 50% 확률로 자동 해제.
- **출력 뉴스**: `"도적 케이가 유적 심층부에서 고대 레릭과 결정적 단서 2개를 안고 안전하게 귀환했다!"`

#### B. 성공 (Normal Success) — 확률 50%
- **자원 획득**: **단서 토큰 1개** 획득, 일반 보급품 아이템 1개 획득.
- **스탯 변화**: HP 손실 정상(10~25 감소), 스태미나 정상 소모(15~30 감소).
- **출력 뉴스**: `"전사 마슈가 유적 하수구 수색을 성공적으로 마치고 귀환했다. (단서 토큰 1 획득)"`

#### C. 실패 (Failure) — 확률 25%
- **자원 획득**: 단서 획득 실패.
- **스탯 변화**: HP 손실 2배(20~40 감소), 스태미나 대량 소모(30~50 감소).
- **카드 변화**: `wounded` (부상) 혹은 `paranoid` (편집증) 상태 카드를 덱에 획득.
- **출력 뉴스**: `"신관 렐리아나가 고대 사원 탐사에 실패하고 부상을 입은 채 탈출했다."`

#### D. 대실패 (Critical Failure) — 확률 10%
- **자원 획득**: 보상 없음.
- **스탯 변화**: **HP가 즉시 1로 고정(중상)**, 스태미나 바닥(0).
- **카드 변화**: `wounded` 카드가 강제 부여되며, 강력한 공포/트라우마 카드인 `panicked` 혹은 `hallucinating` 상태가 확정 부여됩니다.
- **마을 강제 대기**: 치료 및 안정화를 위해 **향후 2일 동안 강제 마을 잔류** 상태가 격발됩니다.
- **출력 뉴스**: `"전사 마슈가 유적에서 끔찍한 실체를 목격하고 미쳐버린 채 간신히 기어 나왔다! 치료가 시급합니다."`

## 4. TownNews 알림 및 플레이어 상호작용 (Player Interactions)

### 4.1 TownNews (도시 소보) 시스템
- 매일 아침(Daily Transition) 또는 특정 시간대가 경과하면, NPA들의 동적 활동 결과들이 취합되어 화면 우측 상단이나 대화 로그에 팝업 또는 큐잉되어 순차 표시됩니다.
- 이 정보를 기반으로 기획자는 유저가 "내가 이 쓸쓸한 멸망의 도시에서 혼자 발버둥 치는 것이 아니구나" 하는 자율적 감각을 제공합니다.

### 4.2 대화 및 단서 교환 (Trade Clue Tokens)
NPA들은 던전에서 대성공/성공을 거두면 고유 자원 공간인 `npc.{id}.clue_tokens`에 단서를 누적해 둡니다. 플레이어는 NPA들을 직접 찾아가 다음과 같이 상호작용할 수 있습니다:

```
[ NPA 대화창 ]
1. "요즘 어떻게 지내?" (상태 확인 / ask_status)
   → NPA가 현재 보유 중인 상태 카드에 맞춰 실시간으로 대사가 변화합니다.
   - (평온) "그냥 그래. 조심히 다녀야지."
   - (편집증) "누가 날 감시하는 것 같아. 너도 나한테 너무 가까이 오지 마..."
   - (환각) "보라색 안개 속에서... 그분이 춤추고 계셔... 하하!"

2. "혹시 확보한 단서를 길드를 위해 공유해 줄 수 있어?" (단서 요청 / request_clue)
   - [성공 조건]: NPA의 신뢰도(Trust)가 일정 수치 이상(보통 gte 3, paranoid 보유 시 gte 6)이고, NPA의 Clue Tokens가 1 이상일 때.
   - [교환 대가]: NPA의 신뢰도가 -2 소모되며, 플레이어의 인벤토리로 단서 토큰(Clue Token) 1개가 양도됩니다.
   - [자동 기부]: 신뢰도가 극도로 높은 NPA(gte 8)는 도시 파멸(Doom Level)이 임계치인 3일 전에 도달하면, 자신이 가진 단서 토큰을 아무 조건 없이 마을 경비대에 자동 기부하여 Doom 게이지를 억제해 줍니다.
```

### 4.3 선물하기 (Gifting System)
- 플레이어는 소지 중인 아이템을 NPA에게 선물하여 신뢰도와 호감도(Affection)를 증강할 수 있습니다.
- **선호 선물 매칭**: NPA가 좋아하는 선호 선물을 건네주면 신뢰도가 3배로 대폭 상승합니다.
- **회복제 증정**: NPA가 부상당했거나 스태미나가 바닥났을 때, 포션이나 진정제 식물을 건네주면 상태가 즉시 치료되며 특수 우호도 대화가 격발됩니다.

## 5. 핵심 NPA 액터 3종 설정

```
┌──────────────────┐      ┌──────────────────┐      ┌──────────────────┐
│  마슈 (Marsh)    │      │ 렐리아나 (Relliana)│    │  케이 (Kay)      │
├──────────────────┤      ├──────────────────┤      ├──────────────────┤
│ - 클래스: 전사   │      │ - 클래스: 신관   │      │ - 클래스: 도적   │
│ - 성향: 공격형   │      │ - 성향: 보조형   │      │ - 성향: 은밀형   │
│ - 위험 던전 선호│      │ - 안전 탐사 위주 │      │ - 고독한 수집가  │
└──────────────────┘      └──────────────────┘      └──────────────────┘
```

### 5.1 전사 마슈 (Marsh) — 돌격형 파이터
- **설명**: 크고 단단한 양손대검을 휘두르는 저돌적인 전사입니다. 험악한 겉모습과 달리 마을을 구하겠다는 투지가 대단합니다.
- **시뮬레이션 프로필**:
  - **스케줄 주 장소**: 선술집, 북쪽 거리
  - **던전 탐사 확률**: 40% (위험 수준이 높은 깊은 던전 선호)
  - **성격**: 무모할 정도로 자주 나가며, 대성공 확률도 높지만 그만큼 실패하여 `wounded` (부상) 카드를 자주 달고 돌아옵니다.
  - **선호 선물**: `에일 맥주(Ale)`, `대형 치료 약초(Healing Herb)`
  - **조우 대사 스타일**: 호탕하고 마초적인 어투를 구사하며, 부상 상태일 때는 끙끙 앓으면서도 괜찮다고 허세를 부립니다.

### 5.2 신관 렐리아나 (Relliana) — 수호 신관
- **설명**: 치료소에서 수련의 엘레나를 돕기도 하는 은빛 십자가의 성스러운 신관입니다.
- **시뮬레이션 프로필**:
  - **스케줄 주 장소**: 치료소, 중앙 광장
  - **던전 탐사 확률**: 15% (매우 신중하며, 정화된 신성 구역 위주로 출입)
  - **성격**: 자율 탐사 성공률이 75%에 육박할 정도로 매우 안정적입니다. 귀환 시 플레이어에게 전파해 주는 `blessed` (축복) 전사 버프 토큰을 자주 가져옵니다.
  - **선호 선물**: `성수(Holy Water)`, `진정 향초(Incense)`
  - **조우 대사 스타일**: 차분하고 경건한 경어로 플레이어를 위로해 주며, 도시의 오염을 크게 우려합니다.

### 5.3 도적 케이 (Kay) — 냉소적인 그림자
- **설명**: 고독하게 혼자 움직이며 고대 유적에서 떨어지는 기계 아티팩트와 결정적 단서를 은밀하게 수집하는 정보통 도적입니다.
- **시뮬레이션 프로필**:
  - **스케줄 주 장소**: 복도, 남쪽 거리의 어두운 구석
  - **던전 탐사 확률**: 55% (기동성을 발휘해 극도로 자주 출입)
  - **성격**: 기동력이 우수해 대실패 확률이 2% 미만으로 극히 적고, 대성공 시 값비싼 보석류를 자주 찾아내어 누적 단서가 매우 많습니다.
  - **제약 조건**: 신뢰도가 극도로 높지 않으면 단서 공유(`request_clue`) 자체를 철저하게 거부합니다.
  - **선호 선물**: `빛나는 루비(Rare Gem)`, `수제 단검(Dagger)`
  - **조우 대사 스타일**: 날카롭고 빈정거리는 조의 냉소적 말투를 지녔으나, 진귀한 선물을 주면 슬그머니 미소를 짓습니다.
